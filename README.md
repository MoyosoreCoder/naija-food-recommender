# NAIJA FOOD RECOMMENDER

## Description
This is a full-stack food recommendation application that helps users find restaurants and addresses in Lagos, Nigeria based on their preference and budget.

## FEATURES
- Helps to discover food in Lagos, Nigeria
- Recommends restaurants based on user's food preferences
- Suggest meals based on user's budget
- Provides restaurant addresses
- Helps users find suitable restaurants to satisfy their cravings

## Tech track
### Backend
- Node js
- Express

### Frontend
- React

- POST  api/recommend
